这个安装包适合 5090d显卡，python 3.10或者3.12，pytorch 2.8 , cuda 12.8，windows 11 操作系统上编译成功。

可能5090显卡也可以使用，但是我没有测试，我自己是5090d 的。

我自己是 windows 11 操作系统，可能 windows 10 也适合，因为查找相关资料和操作系统要求不是很高。
